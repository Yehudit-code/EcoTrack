EcoTrack is a web-based platform designed to support sustainable consumption through data analysis, visualization, and collaboration between individuals and companies.

The system enables users to monitor their resource usage and receive actionable recommendations, while allowing companies to identify high-consumption users within their domain and offer targeted solutions.

---

## System Overview

EcoTrack operates as a dual-sided platform with two primary user roles:

- Individual users
- Companies (service and product providers in sustainability-related domains)

Each role is provided with a dedicated interface and functionality tailored to its needs.

---

## User Side Functionality

The user-facing side of the system focuses on helping individuals reduce their environmental footprint through informed decision-making.

Key features include:

- Consumption data management  
- Data visualization and trend analysis  
- Personalized insights  
- Company proposals  
- Community interaction  

---

## Company Side Functionality

The company-facing side is designed for organizations operating within a specific sustainability category.

Key features include:

- User ranking by consumption  
- Individual user analysis  
- Targeted outreach  
- Interaction tracking  

---

## Role-Based Access Control

EcoTrack implements a role-based architecture that ensures:

- Users and companies are exposed only to relevant data and actions  
- Dedicated interfaces per role  
- Secure separation of responsibilities  

---

## Project Goal

The goal of EcoTrack is to provide a practical, data-driven platform that supports sustainable consumption by connecting individuals with relevant solutions in a structured and transparent manner.
